Это учебный проект (№1?) по JavaFX, в котором пытаюсь изучить его по статьям на хабре (https://habr.com/ru/articles/474292/).

Ну а так же от себя добавляю всякие приколы: от звуков из CS до вечноЧешушийсяОбезьяны.gif :D

На данном этапе реализовано всё до 4-ой статьи, а так же самостоятельно разбирался с rotate, датой (которая обновляется по кнопке), 
прогресс баром, календарём (в процессе выпилил его), чекбоксами, а так же их активацией, открытием ссылки на мой гитхаб

Всё в данном проекте кликабельно: 

1)кнопка "обновить время", при первом нажатии появляется реальное время, которое отображено на компьютере, дальнейшее нажатие обновляет время.

2)кнопка "сменить тему", меняет тему приложения с светлой на тёмную

3)ссылка на гитхаб(гиперссылка), открывает вкладку в браузере и открывает мой гихаб.

4)кнопка "Hello", приветсвует человека.

5)кнопки саундпады - 1,2,3,4. При их активации воспроизводится звук из CS.

6)кнопка "Click me!" - своего рода кликер, при нажатии заполняется прогресс бар и на определённых этапах воспроизводится звук.

7)кнопка "Эта шняга чешется" на голове нашего товарища - включает уникальный звук обезьян.

8)Чекбокс "Режим бога" - увеличивает значение power (которое по дефолту 1) на 99, в результате чего после двух нажатий на "Click me" кликер полностью проходится.

9)Чекбокс "Дебаг режим" - включает особый режим разработчика, который позволяет увидеть значение Power.

10)Чекбоксы "Мод много денег" и "Режим поиска пидорасов" являются шуточными приколами, которые никого не пытались задеть :D
